# Classify
CLASSIFY is a powerful application that provides CRUD functionality and allows users to store and manage a seven-day timetable. With this app, users can easily create, read, update, and delete entries in their timetable, helping them stay organized and plan their schedules effectively. <h1>Features</h1>

<b>Create:</b> Users can create a new timetable entry by specifying the day, time slot, class name, teacher name, and room number. The app provides a user-friendly interface to input this information and  add it to the timetable.

<b>Read:</b>  The app allows users to view their seven-day timetable in a clear and organized manner. Users can easily navigate through different days and time slots to see their scheduled activities at a glance, including the class name, teacher name, and room number for each entry.

<b>Update:</b>  Users can update their timetable entries if there are any changes or modifications to their schedule. The app provides options to edit the day, time slot, class name, teacher name, and room number, ensuring that the timetable stays accurate and up to date.

<b>Delete:</b>  If a scheduled activity or event is no longer relevant or needs to be removed, users can delete the corresponding timetable entry. The app allows for easy deletion, ensuring that the timetable accurately reflects the user's current schedule.
